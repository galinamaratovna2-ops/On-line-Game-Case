# On-line-Game-Case
This case containes the key metrics analysis of the on-line game including financial customer commitment indicators. It has been executed by means of SQL and Excel as post-processing and visualization tool.
Цель проекта: провести анализ ключевых метрик on-line игры: оценить динамику пользовательской базы, активность игроков, финансовые показатели и показатели вирусного роста, оценить влияние маркетинговых акций на финансовые результаты.

Что делала:

- Изучила исходные данные и провёла анализ количества пользователей, игровых сессий и их продолжительность.

- Проанализировала динамику выручки и влияние маркетинговой кампании на финансовые показатели, в том числе на базе концепции факторного анализа.

- Определила коэффициент вирусности игры и средний размер будущих когорт пользователей.

- Оценила изменения ключевых метрик DAU / WAU / MAU и их соотношение.

- Провела когортный анализ пользовательского поведения (на метрике средняя длительность игровой сессии) и финансовых показателей (на метрике ARPU)
- 
- Рассчитала долю лояльных пользователей по разным критериям определения лояльности и провела анализ динамики их количества.
  
- Столкнувшись в ходе анализа с проблемой отсутствия части данных, рассчитала долю проблемных сессий по типам устройств, выдвинула гипотезу о возможных причинах аномалий, предложила вариант решения проблемы.


Используемые инструменты: PostgreSQL (CASE WHEN, агрегации, JOIN-ы, CTE/подзапросы), Excel (постобработка, визуализация).

Результат: подготовлена презентация с основными выводами и визуализациями, демонстрирующая динамику пользовательской активности и влияние маркетинговых факторов на выручку игры, выявлены актуальные и потенциальные проблемы бизнеса, требующие решения. В качестве развития данного проекта можно предложить создание дашборда, позволяющего мониторить ключевые метрики, выявлять проблемные зоны и благодаря возможности более детального анализа в различных срезах оперативно принимать меры к их устранению.
